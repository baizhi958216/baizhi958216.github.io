## 继承
- extends
```dart
void main() {
  MyNormalCar car = MyNormalCar();
  print(car.numberOfSeat);
  
  car.drive();
  
  MyElectricCar electCar = MyElectricCar();
  print(electCar.batteryLevel);
  electCar.reCharge();
  print(electCar.batteryLevel);
  
}

class MyNormalCar {
  int numberOfSeat = 5;
  void drive(){
    print('wheels turn.');
  }
}
// myElectricCar继承了MyNormalCar原有的属性和方法
class MyElectricCar extends MyNormalCar{
  int batteryLevel = 90;
  void reCharge(){
    batteryLevel = 100;
  }
}
```

## 多态
子类的方法覆盖父类的方法  
调用的时候父类没有的方法无法调用  
父类跟子类都有的则可以调用
```dart
class Animal {
  speak() {}
}

class Dog extends Animal {
  @override
  speak() {
    print('wang!');
  }

  run() {
    print('dog run!');
  }
}

class Cat extends Animal {
  @override
  speak() {
    print('miao');
  }

  run() {
    print('cat run!');
  }
}

void main() {
  Animal d1 = Dog();
  Dog d2 = Dog();
  d1.speak();
  //d1.run();无法通过编译 Animal 没有 run() 方法
  d2.speak();
  d2.run();
}
```
