二、环境配置​

1. VisualStudio​

安装即可

2. NodeJS​

安装即可

3. MongoDB​

安装即可

4. Fiddler​

  • 完成安装
  • 依次点击 Tools -> Options
  • 在弹出窗口中点击 HTTPS, 勾选Decrypt HTTPS traffic
  • 依次点击 Actions -> Trust Root Certificate
  • 弹出窗口中选中 Yes -> 是 -> 确定


  • 最后点击 OK

5. 客户端​

安装后等待下载完成资源文件

6. CrepeSR-Resources​

  • 将 StarRail_Data 复制到游戏安装目录的 Games 目录
  • 全部替换
  • 自行新建 data 文件夹到 CrepeSR 的 src目录下
  • 将 proto 和 excel 复制进去

三、启动服务​

MongoDB​

  • 打开终端
  • 新建一个 db 文件夹
mkdir db

  • 进入MongoDB服务目录
cd "C:\Program Files\MongoDB\Server\6.0\bin"

  • 启动MongoDB服务并将数据库挂载
.\mongod.exe --dbpath $HOME/db

CrepeSR​

  • 打开终端
    在 CrepeSR 目录右键打开终端
  • 下载依赖
    运行 npm install
  • 运行服务端
    运行 npm run start
  • 创建一个账号
    格式: account create 用户名 uid 
# 比如
account create starrails 114514

四、设置代理​

  • Fiddler依次点击 Rules -> Customize Rules...
  • 将弹出的窗口内的代码全部替换成以下代码 
import System;
import System.Windows.Forms;
import Fiddler;
import System.Text.RegularExpressions;
class Handlers
{
static function OnBeforeRequest(oS: Session) {
      if(oS.host.EndsWith(".yuanshen.com") || oS.host.EndsWith(".hoyoverse.com") || oS.host.EndsWith(".mihoyo.com") || oS.host.EndsWith(".starrails.com")) {
      oS.host = "localhost";
      }
  }
};

  • 按下键盘 Ctrl + s 保存, 变绿

五、开始游戏​

  • 运行 崩坏 星穹铁道
  • 开始游戏
  • 所有的弹出窗口均点击 yes
  • 密码为任意字符
