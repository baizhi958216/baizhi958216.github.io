## 一、下载HEU_KMS_Activator

:::danger 注意

下载前需要关闭`杀毒软件`(包括Windows自带的`Windows Defender`), 否则HEU_KMS_Activator会被当做恶意软件删除。

关闭方式自行百度。

:::
